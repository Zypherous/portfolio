VR CAMPUS

This project is a prototype of a virtual reality classroom. I made it with the help of three other's in my class. Our goal was to try and create an environment free of distractions where an instructor, students, and others can interact and learn. The project was quite challenging as we wanted to make it as accessible as possible through the use of Google VR. Google VR does not have native hand and gesture recognition. Using Manomotion, and coding it to work with Google VR allowed for some buggy but somewhat functional gesture recognition. This allowed the creation of a whiteboard where a user can grab a "pen" and write on the board. I will return to this project when I have time and my professor is willing to be my mentor for that project! Feel free to take a look at the documentation I created for this project here.
